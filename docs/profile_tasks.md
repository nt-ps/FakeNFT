# Profile Epic Decomposition

## Profile Screen

- Экран Профиль - 10ч
  - Общее вью с инфо о профиле (аватарка, имя, описание) - 3ч
  - Таблица для перехода на экраны (Мои NFT, Избранные NFT) - 2ч
  - Загрузка информации о пользователе - 5ч
  - Кнопка редактировать + открытие экрана Редактирования профиля - 1ч
    
- Экран Редактирование профиля - 11ч
  - UI с инфо о пользователе - 3ч
  - Изменение аватарки - 3ч
  - Сохранение инфо о пользователе - 5ч
  - Закытие экрана - 1ч
